### **Interactive Tableau Dashboard | Labour Market Analysis | EMSI UK Dataset**

This project visualises employment trends across the UK from **2011 to 2014** using EMSI (Economic Modeling Specialists International) labour-market data.  
The dashboard offers insights into job growth, industry performance, and year-over-year changes across multiple regions.
